# Automotive Sales Analysis Dashboard (Power BI)

## Project Overview 
This project analyzes automotive parts sales data using Microsoft Power BI to uncover insights related to product performance, warehouse efficiency, and sales trends.

The dashboard helps answer important business questions such as:
- Which product lines generate the most revenue?
- Which warehouse performs best in sales and quantity?
- What is the average revenue per order?
- How do product demand and revenue compare?

The goal of this project is to demonstrate data analysis, visualization, and business insight generation using Power BI.

----

##  Dataset Information (sales.xlsx)
The dataset contains transactional sales data for automotive parts.

### Key Columns
| Column        | Description                          |
|---------------|--------------------------------------|
| Order_Number  | Unique identifier for each order     |
| Date          | Order date                           |
| Product_Line  | Category of automotive product       |
| Quantity      | Number of units sold                 |
| Unit_Price    | Price per unit                       |
| Total         | Total order value                    |
| Payment       | Payment method                       |
| Payment_Fee   | Transaction fee percentage           |
| Warehouse     | Warehouse fulfilling the order       |

##  Data Processing
The dataset was prepared and analyzed in Power BI.

## Page 1 - Executive Summary 
High level summary of sales performance. 
Key visuals: 
- Total Net Revenue KPI
- Total Orders KPI

## Page 2 - Product & Warehouse Performance 
Operational Insights into products and warehouses.
Visualizations: 
1. Top Product Lines by Revenue
2. Quantity Sold by Product Line
3. Warehouse Sales Comaparison
4. Revenue per Order by Warehouse

## Page 3 - Customer & Payment Insights
1. Revenue By Client Type
2. Revenue By Payment
3. Payment Method Distribution
4. Payment Fee Impact

## Key Insights 
Some insights from the analysis:
Suspension & Traction and Frame & Body generate the highest revenue.
The Central warehouse handles the largest sales volume.
North warehouse shows competitive performance in order value.
Certain product lines sell fewer units but produce higher revenue.
